# Java Data Types

Data types is a type of data that a variable can store.

## Datatype Syntax

'''java
datatype variable_name = variable_value;
'''

## Datatype Example

'''java
int myAge = 29;
'''

# Types Of Data types

## Primitive Data Types

Primitive data types are used to store single value

- byte
- short
- int
- long
- float
- double
- char
- boolean


## Non Primitive Data Types

- String
- Array
